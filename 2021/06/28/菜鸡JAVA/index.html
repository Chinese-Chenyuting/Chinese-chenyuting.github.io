java的组成单元是类 而类又包括{属性 和方法} 一个java程序有许多类组成
1.类的属性称为类的全局变量 全局变量声明在类中/方法的属性称为局部变量 局部变量声明在方法体中
2.如何导入类呢?这是一个问题
在C语言中，一般都是通过#include<stdio.h>来导入
而在JAVA中，一般导入API类库，通过import类关键字导入相关的类
3public static void 分别为main（）的权限修饰符 静态修饰符 返回值修饰符 这些都是固定的
